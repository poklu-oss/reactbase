
Style Framework :  https://material-ui.com/components/box/
UI Framework    : ReactJS

Installation:
  -  Install Nodejs 

Dev Setup:
  $ npm install

Run Application:
  $ npm start

Build Production Distribution:
   $ npm run-script build

 Build Docker Image:
    $ docker build . -t reactbase

 Run Docker:
    $ docker run -p 8080:80  reactbase

 Open : http://localhost:8080
